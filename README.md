# Generic-Dynamic-Data-Structures
C++ implementation of generic dynamic data structures (dynamic array, stack, queue) using templates, inspired by STL-like behavior.


# 📦 Dynamic Data Structures in C++ (Generic Implementation)

## 📌 Overview
This project is a C++ implementation of fundamental dynamic data structures including Dynamic Array, Stack, and Queue using **templates (generic programming)**.

The goal of the project is to simulate STL-like behavior from scratch without using built-in structures, while focusing on memory management, performance, and clean OOP design.

---

## 🛠️ Technologies Used
- C++
- Object-Oriented Programming (OOP)
- Templates (Generic Programming)
- Dynamic Memory Allocation
- Console Application

---

## 🧱 Implemented Data Structures

### 📊 Dynamic Array
- Auto resizing mechanism
- Insert / Delete / Access operations
- Memory-efficient resizing strategy

### 📚 Stack
- LIFO (Last In First Out) implementation
- Push / Pop / Peek operations
- Built on dynamic array

### 📥 Queue
- FIFO (First In First Out) implementation
- Enqueue / Dequeue operations
- Efficient element management

---

## ⚙️ Key Features
- Fully generic using C++ templates
- No STL containers used (from scratch implementation)
- Memory-efficient dynamic allocation
- Structured like simplified STL behavior
- Clean separation between data structures
